# liturgia-diaria
API que fornece as orações e leituras do dia da Santa Missa (liturgia dária).
> Acesse pelo link: http://liturgia.herokuapp.com
> Caso deseje ir a um dia específico, http://liturgia.herokuapp.com/[dia]-[mês] <br>
> (O dia sem o 0 caso seja menor que 10)
