# liturgia-diaria
API que fornece as orações e leituras do dia da Santa Missa (liturgia dária).
> Acesse pelo link: http://liturgia.herokuapp.com <br>
> 
> Caso deseje ir a um dia específico, http://liturgia.herokuapp.com/[dia]-[mês]
> (O dia sem o 0 caso seja menor que 10)

> Ou:
> http://liturgia.herokuapp.com/?dia=[dia]&mes=[mês]
