# liturgia-diaria
API que fornece as orações e leituras do dia da Santa Missa (liturgia dária).
> Acesse pelo link: https://liturgiatododia.herokuapp.com/ <br>
> 
> Caso deseje ir a um dia específico, https://liturgiatododia.herokuapp.com/[dia]-[mês] <br>
> (O dia sem o 0 caso seja menor que 10)

> Ou:
> https://liturgiatododia.herokuapp.com/?dia=[dia]&mes=[mês] <br>
> Caso o mês não seja definido, o padrão será o mês atual.
